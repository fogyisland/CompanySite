B

GitHub 风 — 信息密集

侧栏 232px、表格高密度、表单紧凑。多列、单页信息量大。给"天天看 200 行订单"的人。

主色 #0969da   字号 12-13px   行高 32px
C

Stripe 风 — 商务留白

侧栏 240px、内容宽留白、卡片化、统计数字大。给"看仪表盘决策"的人。

主色 #635bff   字号 14-15px   行高 48px
D

Ant Design Pro 风 — 企业级

侧栏深色 208px + 顶部面包屑 + 标签页 + 多级菜单。表单表格表单感强。给"管 N 个模块"的人。

主色 #1677ff   字号 13-14px   行高 40px
