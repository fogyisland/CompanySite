后台 UI 风格对比

当前 admin-product.html 等页面用的是通用模板，间距大、样式偏卡通。下面 4 个候选都更"商务"。看哪个方向对，再细化细节。

A

Linear / Vercel 风 — 极简深色

侧边栏窄（220px）、字小、间距紧凑。深灰背景 + 微弱色块。表格用极细分隔线。代表：SaaS 开发者工具。专业但视觉安静。

GitHub / GitLab 风 — 信息密集
