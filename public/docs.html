下载安装

系统要求

在安装我们的软件之前，请确保您的系统满足以下要求：

  • Windows: Windows 10/11 (64位)
  • Mac: macOS 10.14 或更高版本
  • Linux: Ubuntu 18.04、Debian 10 或类似发行版

下载软件

  1. 登录您的账户
  2. 进入"用户中心"
  3. 找到您购买的产品
  4. 点击"下载"按钮获取安装包

提示：下载链接也会发送到您的注册邮箱，方便您随时重新下载。

安装步骤

  1. 双击下载的安装包（.exe/.dmg/.deb）
  2. 按照安装向导提示完成安装
  3. 选择安装目录（建议使用默认目录）
  4. 点击"完成"启动软件

激活授权

获取授权密钥

购买成功后，您可以通过以下方式获取授权密钥：

  • 购买确认邮件中的密钥
  • 用户中心的订单详情页

激活软件

  1. 启动已安装的软件
  2. 点击"激活"或"输入密钥"
  3. 粘贴您的授权密钥
  4. 点击"激活"完成授权

XXXX-XXXX-XXXX-XXXX

常见激活问题

密钥无效？ 请检查密钥是否完整复制，或联系客服获取帮助。

已达到激活次数？ 我们的软件允许在多台设备上激活（根据许可证类型），如需转移激活，请先在原设备上注销。

账户管理

注册账户

访问用户注册页面，填写用户名、密码即可完成注册。

找回密码

  1. 访问登录页面
  2. 点击"忘记密码"
  3. 输入您的注册邮箱
  4. 查收重置邮件并设置新密码

修改个人信息

登录后进入用户中心，您可以修改：

  • 个人资料信息
  • 登录密码
  • 邮箱地址

支付发票

支付方式

我们支持以下支付方式：

  • 微信支付
  • 支付宝
  • 银行卡支付

开具发票

如需开具发票，请在购买后联系客服，提供以下信息：

  • 订单编号
  • 发票抬头（个人/公司）
  • 税号（如需增值税发票）
  • 接收邮箱

退款流程

在30天退款保障期内，您可以申请全额退款：

  1. 联系客服提交退款申请
  2. 提供订单信息和退款原因
  3. 审核通过后，退款原路返回
  4. 等待1-3个工作日到账

更新升级

自动更新

软件会在有新版本时自动提示更新。您也可以手动检查更新：

  1. 打开软件设置
  2. 点击"检查更新"
  3. 如有新版本，点击"下载更新"
  4. 重启软件完成升级

版本升级

购买我们的软件后，在同一主版本周期内，您可以免费获得所有更新。例如，购买 v2.0 版本后，可以免费升级到 v2.1、v2.2 等。

提示：重大版本升级（如 v2.0 到 v3.0）可能需要另行付费。

卸载重装

Windows 卸载

  1. 打开"设置" > "应用"
  2. 找到要卸载的软件
  3. 点击"卸载"
  4. 按照提示完成卸载

Mac 卸载

  1. 打开"访达" > "应用程序"
  2. 找到要卸载的软件
  3. 将软件拖到"废纸篓"
  4. 清空废纸篓

重新安装

卸载后您可以随时重新安装。使用原来的授权密钥激活即可继续使用。
